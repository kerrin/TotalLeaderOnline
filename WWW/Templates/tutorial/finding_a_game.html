How To Find a Game

First thing we need to do is find a game to join.
To find a game, select the Find Game button on the left menu. For now just press the Find button to list all games that have yet to start, later you can fill out any requirements for a game you would like to play, and then press the Find button to find a game that matches your playing needs.

Leaving fields blank and the drop down boxes on any will not restict the search on that crieria.

After pressing the Find button you will be presented with the list of games. Note: If no games are listed, then there are no games waiting to start, so look in the Manual to see how to create a game.

To view a game, select it by pressing the Select button on the result screen.

Choosing a Game

Once a game has been selected you will be taken to the board for that game. To return to the board of the cuurently selected game at anytime, simply press the Board button on the left hand menu. If the menu does not have a board button, you currently have no selected game.

For move details on finding a creating games, veiw the Getting Started section of the Manual
