Making a Move

To move you will need units on the current square, select the direction you want to move the units in by changing the drop down or pressing the appropriate square on the area summary.
Next select the number of units to move by changing the drop down.
Next make sure the move type drop down is set to move.
Once you are happy with the move press the Move button.
You can build expansions on major squares and start effects on any square by changing the move type drop down.

If you change your mind, and wish to cancel a move, simply move 0 units to the square the move is for, and the move will be cancelled

Lets move the 9 units on the major square. So view the square in the center of you land at the center of the cross.
Next view the surrounding squares by hovering the mouse over them, find one owned by the natives that has only a few units on and select it by pressing the mouse button. The direction drop down will now automatically change to the correct direction.
The number of units should already be set to 9, if not change it to 9.
The move type should already be set to move, if not change it to move.
Now press the move button. The units should now move from the center square to the selected square, this shows where the units are going to move to and the number that are moving.

These units will attack the native units when the next turn occurs. For an explaination of how fighting works view the appropriate section of the Manual

Now we have moved the units from this square, we have made space for more units to come in. We could move units from a neighbouring square if we wanted to, but since this is our base we can assign ne recruits here.

To assign recruits, make sure the recruits drop down is set to 9 (note the maximum number of recruits in the drop down is the number you can fit in the base, or how many you still have to assign, which ever is lower.)
Next press the rectruit button and the recruits are assigned.
You can cancel assigning recruits by setting the number to 0 and pressing assign.

Other types of move can also be made from this screen, and these are outlined in later sections of the tutorial.
