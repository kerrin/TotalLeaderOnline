Joining Someone Elses Game

It is always recommended that players look for existing games waiting to start that match their requirements before creating a new game.

To find a game, select the Find Game button on the left menu. Fill out any requirements for a game you need, and then press the Find button.

Leaving fields blank and the drop down boxes on any will not restict the search on that crieria. To find all games not started yet, simply press the Find Game button without changing anything

By default (if you change nothing before pressing the Find button) it will find all games that are waiting to start.

After pressing the Find button you will be presented with the list of games matching your requirements. If none are shown, no games fitting your requirements were found, so go back and reduce your requirements, or create a new game

To view a game, select it by pressing the Select button on the result screen.

Choosing a Game

Once a game has been selected you will be taken to the board for that game. To return to the board of the cuurently selected game at anytime, simply press the Board button on the left hand menu. If the menu does not have a board button, you currently have no selected game.

You can view the players currently in the game by pressing the Players button on the left menu

Starting a New Game

If you cannot find a game you wish to join, you can create a new game.

To create a game simply press the New Game button on the left menu and fill out the form to specify the game parameters. Once you are happy with the game parameters, press the Create Game button.

Creating a game will automatically select it as your current game.

Please note, that you are not guarenteed a position in a game you have created, but if a game does fill up before you join it, it is clearly very popular, as create another with the same criteria.

ParameterMeaning
Game NameA name for the game, used in messaging and game lists
Number of PlayersMaximum number of players allowed in the game.
Note: The game will still start even if this number of players is not reached.
Only games with less than 2 players are not started.
Board WidthThe number of squares the entire game board is wide
Board HeightThe number of squares the entire game board is high
Turn FrequencyHow often a turn will occur during the games active time
WeekdaysThe days that the game turns will happen on
Start Turns Daily AtThe hour that the game becomes active each day it is active on (in 24 hour GMT)
Stop Turns Daily AtThe hour that the game stops being active each day (in 24 hour GMT)
Start Game OnThe date the first turn will happen on.
It is recommended that games are started 1 to 7 days after their creation, to allow players to join.
Starting a game in the past will immediatly disable it.
Maximum Game TurnsThe number of turns the game can have before it will end, with the winner being decided by scoring
